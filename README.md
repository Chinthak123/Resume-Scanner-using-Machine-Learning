# Resume-Scanner-using-Machine-Learning
Project Overview

This project is a Resume Scanner built using Python and machine learning techniques. The main aim is to analyze resumes and extract meaningful information to understand how well a resume matches a particular job role or requirement. It automates the initial resume screening process and reduces manual effort.
